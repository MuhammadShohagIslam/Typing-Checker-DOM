Difference Between Local Storage, Session Storage

Local Storage:Local Storage is a one kind of database where we can store data. It's a permanently data storage place


Session Storage:Session Storage is a one kind of database where we can store data, but after browser closed We can find get data. It's a temporaray data storage place

How Javascript Event Loop Works

The event loop has a simple task: monitor the call stack and callback queues. If the call stack is empty, the event loop takes the first event from the queue and pushes it onto the call stack, where it executes. Such an iteration is called a tick in the event loop.
